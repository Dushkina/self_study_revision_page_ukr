Розкриття дужок
з коренями · Surds

Expanding brackets containing surds (ірраціональні числа). Правило FOIL, квадрат суми/різниці, різниця квадратів — з прикладами та практикою.

Ключ · Key identity
\( \sqrt{a}\cdot\sqrt{a} = (\sqrt{a})^2 = a \)
FOIL Rule
\( (a+b)(c+d) = ac+ad+bc+bd \)
§1 · Основи · Key Rules

Що потрібно знати перед розкриттям дужок

Зведення кореня у квадрат · Squaring a surd
√a · √a = a
\((\sqrt{a})^2 = \sqrt{a}\cdot\sqrt{a} = a\)
\((k\sqrt{a})^2 = k^2\cdot(\sqrt{a})^2 = k^2\cdot a\)
\((2\sqrt{3})^2 = 4\cdot 3 = 12\quad (3\sqrt{5})^2 = 9\cdot 5 = 45\)
Множення коренів · Multiplying surds
√a · √b = √(ab)
\(\sqrt{a}\cdot\sqrt{b} = \sqrt{ab}\)
\(\sqrt{2}\cdot\sqrt{5} = \sqrt{10}\quad 3\sqrt{2}\cdot 4\sqrt{2} = 12\cdot 2 = 24\)
Подібні доданки · Like surds
collect terms
\(3\sqrt{2}+5\sqrt{2} = 8\sqrt{2}\quad 7\sqrt{3}-2\sqrt{3} = 5\sqrt{3}\)

Як алгебраїчні змінні: \(\sqrt{2}\) та \(\sqrt{3}\) — різні, не можна спростити \(\sqrt{2}+\sqrt{3}\)

§2 · Метод FOIL · FOIL Method
⚡ Правило FOIL · The FOIL Rule

Для двочленів: (a+b)(c+d) перемножуємо 4 пари:

First (перші) · Outer (зовнішні) · Inner (внутрішні) · Last (останні)

\((a+b)(c+d) = \underbrace{ac}_{\text{F}} + \underbrace{ad}_{\text{O}} + \underbrace{bc}_{\text{I}} + \underbrace{bd}_{\text{L}}\)

Потім зібрати подібні доданки (collect like terms).

Алгоритм · Algorithm

01

Записати 4 добутки · Write 4 products (FOIL)

Кожен член першої дужки множимо на кожен член другої.

\((2+\sqrt{2})(3+\sqrt{2}) = 2\cdot3+2\sqrt{2}+3\sqrt{2}+\sqrt{2}\cdot\sqrt{2}\)
02

Спростити добутки коренів · Simplify surd products

Замінити \(\sqrt{a}\cdot\sqrt{a}=a\), \(\sqrt{a}\cdot\sqrt{b}=\sqrt{ab}\) тощо.

\(= 6 + 2\sqrt{2} + 3\sqrt{2} + 2\)
03

Зібрати подібні доданки · Collect like terms

Числа окремо, корені окремо.

\(= (6+2) + (2+3)\sqrt{2} = 8 + 5\sqrt{2}\)
§3 · Важливі шаблони · Key Patterns

Три спеціальні форми

Квадрат суми · Perfect square (sum)
\((a+b)^2 = a^2+2ab+b^2\)
\((a+\sqrt{b})^2 = a^2+2a\sqrt{b}+b\)

Приклад: \((3+\sqrt{2})^2 = 9+6\sqrt{2}+2 = 11+6\sqrt{2}\)

Квадрат різниці · Perfect square (diff)
\((a-b)^2 = a^2-2ab+b^2\)
\((a-\sqrt{b})^2 = a^2-2a\sqrt{b}+b\)

Приклад: \((3-\sqrt{2})^2 = 9-6\sqrt{2}+2 = 11-6\sqrt{2}\)

Різниця квадратів · Difference of squares
\((a+b)(a-b) = a^2-b^2\)
\((\sqrt{a}+\sqrt{b})(\sqrt{a}-\sqrt{b}) = a-b\)

Результат — раціональне число! Це основа раціоналізації.

Приклад: \((3+\sqrt{2})(3-\sqrt{2}) = 9-2 = 7\)

Квадрат \((k\sqrt{a})^2\) · Squaring
\((k\sqrt{a})^2 = k^2\cdot a\)

Приклади:

\((2\sqrt{3})^2 = 4\cdot 3 = 12\)

\((3\sqrt{5})^2 = 9\cdot 5 = 45\)

\((7\sqrt{10})^2 = 49\cdot 10 = 490\)

§4 · Розібрані приклади · Worked Examples
Приклад 1 · FOIL — базовий
\((2+\sqrt{2})(3+\sqrt{2})\)
①FOIL: \(2\cdot3 + 2\cdot\sqrt{2} + \sqrt{2}\cdot3 + \sqrt{2}\cdot\sqrt{2}\)чотири добутки
②\(= 6 + 2\sqrt{2} + 3\sqrt{2} + 2\)\(\sqrt{2}\cdot\sqrt{2}=2\)
✓\(= 8 + 5\sqrt{2}\)collect like terms
Приклад 2 · FOIL — від'ємний член
\((3+\sqrt{5})(1-\sqrt{5})\)
①FOIL: \(3\cdot1 + 3\cdot(-\sqrt{5}) + \sqrt{5}\cdot1 + \sqrt{5}\cdot(-\sqrt{5})\)уважно зі знаком мінус
②\(= 3 - 3\sqrt{5} + \sqrt{5} - 5\)\(\sqrt{5}\cdot\sqrt{5}=5\)
✓\(= -2 - 2\sqrt{5}\)collect like terms
Приклад 3 · Квадрат суми
\((3+\sqrt{2})^2\)
①\((3+\sqrt{2})(3+\sqrt{2}) = 9 + 3\sqrt{2} + 3\sqrt{2} + (\sqrt{2})^2\)FOIL або шаблон \((a+b)^2\)
②\(= 9 + 6\sqrt{2} + 2\)\((\sqrt{2})^2=2\)
✓\(= 11 + 6\sqrt{2}\)
Приклад 4 · Різниця квадратів
\((\sqrt{7}+2)(\sqrt{7}-3)\)
①FOIL: \((\sqrt{7})^2 - 3\sqrt{7} + 2\sqrt{7} - 6\)не є точна різниця квадратів (різні числа)
②\(= 7 - 3\sqrt{7} + 2\sqrt{7} - 6\)\((\sqrt{7})^2=7\)
✓\(= 1 - \sqrt{7}\)collect like terms
Приклад 5 · З числом перед коренем
\((3\sqrt{2}+1)(2\sqrt{2}-2)\)
①FOIL: \(3\sqrt{2}\cdot2\sqrt{2} + 3\sqrt{2}\cdot(-2) + 1\cdot2\sqrt{2} + 1\cdot(-2)\)multiply all pairs
②\(= 6\cdot2 - 6\sqrt{2} + 2\sqrt{2} - 2 = 12 - 6\sqrt{2} + 2\sqrt{2} - 2\)\(3\sqrt{2}\cdot2\sqrt{2}=6\cdot(\sqrt{2})^2=12\)
✓\(= 10 - 4\sqrt{2}\)collect like terms
Приклад 6 · Точна різниця квадратів
\((6-2\sqrt{2})(2+\sqrt{2})\)
①FOIL: \(6\cdot2 + 6\sqrt{2} + (-2\sqrt{2})\cdot2 + (-2\sqrt{2})\cdot\sqrt{2}\)уважно зі знаком
②\(= 12 + 6\sqrt{2} - 4\sqrt{2} - 2\cdot2 = 12 + 6\sqrt{2} - 4\sqrt{2} - 4\)\(-2\sqrt{2}\cdot\sqrt{2}=-2\cdot2=-4\)
✓\(= 8 + 2\sqrt{2}\)collect like terms
