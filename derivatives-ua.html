Означення похідної · Definition of Derivative
$$f'(x) = \lim_{\Delta x \to 0} \frac{f(x+\Delta x)-f(x)}{\Delta x}$$
§1 · Означення та геометричний зміст · Definition

Що таке похідна?

Похідна функції \(f(x)\) в точці \(x_0\) — це границя відношення приросту функції \(\Delta y\) до приросту аргументу \(\Delta x\), коли \(\Delta x \to 0\).

Означення · Definition
limit
$$f'(x_0) = \lim_{\Delta x \to 0} \frac{\Delta y}{\Delta x} = \lim_{\Delta x \to 0} \frac{f(x_0+\Delta x)-f(x_0)}{\Delta x}$$
Геометричний зміст · Geometric meaning
tangent

Похідна \(f'(x_0)\) — нахил (кутовий коефіцієнт) дотичної до графіка в точці \((x_0,\,f(x_0))\).

\(\text{Рівняння дотичної:} \quad y = f(x_0) + f'(x_0)(x - x_0)\)
\(\text{Рівняння нормалі:} \quad y = f(x_0) - \dfrac{1}{f'(x_0)}(x - x_0)\)
Фізичний зміст · Physical meaning
physics
\(s'(t) = v(t)\) — миттєва швидкість (instant velocity)
\(v'(t) = a(t)\) — миттєве прискорення (acceleration)

§3 · Властивості · Properties

Основні властивості похідної

Лінійність · Linearity
linear
\((af + bg)' = af' + bg', \quad a,b = \text{const}\)
\((C)' = 0\) — похідна константи дорівнює нулю
Power Rule · Степенева функція
power
\((x^n)' = n\,x^{n-1}\) — для будь-якого дійсного \(n\)
\((x^2)'=2x, \quad (x^3)'=3x^2, \quad (\sqrt{x})'=\dfrac{1}{2\sqrt{x}}, \quad \left(\dfrac{1}{x}\right)'=-\dfrac{1}{x^2}\)
Неперервність · Continuity
continuity

Якщо функція має похідну в точці \(x_0\), то вона неперервна в цій точці. Зворотне твердження хибне.

§4 · Правила диференціювання · Differentiation Rules
Сума і різниця · Sum & Difference
$$(u \pm v)' = u' \pm v'$$

Похідна суми (різниці) = сума (різниця) похідних.

Product Rule · Правило добутку
$$(uv)' = u'v + uv'$$

де \(u=u(x)\), \(v=v(x)\). Похідна добутку двох функцій.

Quotient Rule · Правило частки
$$\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$$

де \(v \neq 0\). «Верхня' × нижня − верхня × нижня'» / нижня².

Chain Rule · Правило ланцюжка
$$\frac{dy}{dx} = \frac{dy}{du}\cdot\frac{du}{dx}$$

де \(y=f(u)\), \(u=g(x)\). Похідна зовнішньої × похідна внутрішньої.

Обернена · Inverse function
$$\frac{dy}{dx} = \dfrac{1}{\;\dfrac{dx}{dy}\;}$$

Якщо \(y=f(x)\) строго монотонна і диференційовна.

Що таке sec, csc (cosec), cot (ctg)? · Reciprocal functions
визначення

Це обернені тригонометричні функції (reciprocal trig functions) — не плутати з оберненими (arcsin/arccos/arctan)!

\(\sec x = \dfrac{1}{\cos x}\) — секанс (secant). Не визначений, коли \(\cos x = 0\)
\(\csc x = \operatorname{cosec}\, x = \dfrac{1}{\sin x}\) — косеканс (cosecant). Не визначений, коли \(\sin x = 0\)
\(\cot x = \operatorname{ctg}\, x = \dfrac{\cos x}{\sin x} = \dfrac{1}{\tan x}\) — котангенс (cotangent). Не визначений, коли \(\sin x = 0\)

Тому \((\sec x)' = \left(\dfrac{1}{\cos x}\right)'\) — можна вивести через правило частки → отримаємо \(\sec x\tan x\).

Пам'ятка · Reminder
chain rule

Для кожної формули вище: спочатку диференціюємо зовнішню функцію (залишаючи \(u\) як аргумент), потім множимо на \(u'\). Це і є правило ланцюжка.

\(\bigl[f(u(x))\bigr]' = \underbrace{f'(u)}_{\text{зовнішня}} \cdot \underbrace{u'}_{\text{внутрішня}}\)
§9 · Розібрані приклади · Worked Examples
Приклад 1 · Правило ланцюжка — степінь
\(y = 4(5x^3+2)^6\)
①Зовнішня: \(u^6\), внутрішня: \(v=5x^3+2\)визначаємо структуру
②\(u'=24v^5,\quad v'=15x^2\)диференціюємо кожну
③\(y'=24(5x^3+2)^5\cdot 15x^2\)chain rule
✓\(y' = 360x^2(5x^3+2)^5\)спрощуємо
Приклад 2 · Корінь від функції
\(y = \sqrt{9x-4}\)
①\(y=(9x-4)^{1/2}\), зовнішня: \(\square^{1/2}\), внутрішня: \(9x-4\)записуємо як степінь
②\(y'=\tfrac{1}{2}(9x-4)^{-1/2}\cdot 9 = \dfrac{9}{2\sqrt{9x-4}}\)chain rule
✓\(y' = \dfrac{9}{2\sqrt{9x-4}}\)
Приклад 3 · Логарифм від тригонометрії
\(y = \ln(\cos x)\)
①Зовнішня: \(\ln(\square)\), внутрішня: \(\cos x\)шаблон \(\ln(g(x))\)
②\(y'=\dfrac{1}{\cos x}\cdot(-\sin x)\)\((\ln\square)'=1/\square\), \((\cos x)'=-\sin x\)
✓\(y' = -\dfrac{\sin x}{\cos x} = -\tan x\)спрощуємо
Приклад 4 · Правило добутку
\(y = x^2 e^{3x}\)
①\(u=x^2,\quad v=e^{3x}\)виділяємо множники
②\(u'=2x,\quad v'=3e^{3x}\)диференціюємо кожен
③\(y'=u'v+uv'=2x\cdot e^{3x}+x^2\cdot 3e^{3x}\)product rule
✓\(y' = e^{3x}(2x+3x^2) = xe^{3x}(2+3x)\)виносимо спільний множник
Приклад 5 · Правило частки
\(y = \dfrac{\sin x}{x^2}\)
①\(u=\sin x,\quad v=x^2\)чисельник і знаменник
②\(u'=\cos x,\quad v'=2x\)
③\(y'=\dfrac{u'v-uv'}{v^2}=\dfrac{\cos x\cdot x^2-\sin x\cdot 2x}{x^4}\)quotient rule
✓\(y' = \dfrac{x\cos x - 2\sin x}{x^3}\)ділимо на x
Приклад 6 · Reciprocal trig · 2a
\(y = \cot 4x\)
①Шаблон: \((\cot u)'=-\dfrac{u'}{\sin^2 u}\), де \(u=4x\)chain rule для ctg
②\(u'=(4x)'=4\)похідна внутрішньої
✓\(y' = -\dfrac{4}{\sin^2 4x}\)підставляємо
Приклад 7 · Reciprocal trig · 2c
\(y = \csc 4x\)
①Шаблон: \((\csc u)'=-\csc u\cot u\cdot u'\), де \(u=4x\)chain rule для cosec
②\(u'=4\)похідна внутрішньої
✓\(y' = -4\csc 4x\cot 4x\)підставляємо
Приклад 8 · Reciprocal trig · 2d — подвійний ланцюжок
\(y = \sec^2 3x\)
①\(y = (\sec 3x)^2\) — зовнішня: \(\square^2\), внутрішня: \(\sec 3x\)виділяємо структуру
②Chain rule перший рівень: \(y'=2\sec 3x\cdot(\sec 3x)'\)power rule
③\((\sec 3x)'=\sec 3x\tan 3x\cdot 3\) — другий рівень ланцюжка\((\sec u)'=\sec u\tan u\cdot u'\)
④\(y'=2\sec 3x\cdot 3\sec 3x\tan 3x\)множимо
✓\(y' = 6\sec^2 3x\tan 3x\)спрощуємо
Приклад 9 · Reciprocal trig · 3b — корінь
\(y = \sqrt{\cot x}\)
①\(y=(\cot x)^{1/2}\) — зовнішня: \(\square^{1/2}\), внутрішня: \(\cot x\)записуємо як степінь
②\((\cot x)'=-\dfrac{1}{\sin^2 x}\)похідна cot x
③\(y'=\dfrac{1}{2}(\cot x)^{-1/2}\cdot\left(-\dfrac{1}{\sin^2 x}\right)\)chain rule
✓\(y' = -\dfrac{1}{2\sin^2 x\,\sqrt{\cot x}}\)відрицательний знак зберігаємо
Приклад 10 · Reciprocal trig · 3e — степінь
\(y = \sec^3 x\)
①\(y=(\sec x)^3\) — зовнішня: \(\square^3\), внутрішня: \(\sec x\)виділяємо структуру
②\((\sec x)'=\sec x\tan x\)похідна sec x
③\(y'=3(\sec x)^2\cdot\sec x\tan x\)chain rule: \(3\square^2\cdot(\sec x)'\)
✓\(y' = 3\sec^3 x\tan x\)спрощуємо
Приклад 11 · Inverse trig · 9b
\(y = \arctan\!\left(\dfrac{x}{2}\right)\)
①Шаблон: \((\arctan u)'=\dfrac{u'}{1+u^2}\), де \(u=\dfrac{x}{2}\)inverse trig chain rule
②\(u'=\dfrac{1}{2}\)похідна \(\tfrac{x}{2}\)
③\(y'=\dfrac{\tfrac{1}{2}}{1+\left(\tfrac{x}{2}\right)^2}=\dfrac{\tfrac{1}{2}}{1+\tfrac{x^2}{4}}\)підставляємо
④\(=\dfrac{\tfrac{1}{2}}{\tfrac{4+x^2}{4}}=\dfrac{1}{2}\cdot\dfrac{4}{4+x^2}\)спільний знаменник
✓\(y' = \dfrac{2}{4+x^2}\)спрощуємо
Приклад 12 · Inverse trig · 9c
\(y = \arcsin 3x\)
①Шаблон: \((\arcsin u)'=\dfrac{u'}{\sqrt{1-u^2}}\), де \(u=3x\)inverse trig chain rule
②\(u'=3\)похідна \(3x\)
✓\(y' = \dfrac{3}{\sqrt{1-(3x)^2}} = \dfrac{3}{\sqrt{1-9x^2}}\)підставляємо
Приклад 13 · Inverse trig · 9h
\(y = \arccos x^2\)
①Шаблон: \((\arccos u)'=-\dfrac{u'}{\sqrt{1-u^2}}\), де \(u=x^2\)inverse trig chain rule
②\(u'=2x\)похідна \(x^2\)
✓\(y' = -\dfrac{2x}{\sqrt{1-x^4}}\)підставляємо \((x^2)^2=x^4\)
Приклад 14 · Inverse trig + exp · 9l — подвійний ланцюжок
\(y = e^{\arctan x}\)
①Зовнішня: \(e^{\square}\), внутрішня: \(u=\arctan x\)виділяємо структуру
②\((e^u)'=e^u\cdot u'\)шаблон exp
③\(u'=(\arctan x)'=\dfrac{1}{1+x^2}\)похідна arctan x
✓\(y' = e^{\arctan x}\cdot\dfrac{1}{1+x^2} = \dfrac{e^{\arctan x}}{1+x^2}\)підставляємо
