When to use \(B(n,\,p)\)

Model X with a Binomial Distribution

\(B(n,\,p)\)
English
  • Fixed number of trials, n
  • Two possible outcomes — success or failure
  • Fixed probability of success, p
  • Trials are independent of each other
Українська
  • Фіксована кількість випробувань n
  • Два можливих результати — успіх і невдача
  • Ймовірність успіху p однакова у кожному випробуванні
  • Усі випробування незалежні одне від одного
Core Formulas
Probability of exactly r successes
\(P(X = r) = \binom{n}{r} p^r (1-p)^{n-r}\)
n = index (number of trials)
p = parameter (probability of success)
r = number of successes
Combination coefficient
\(\dbinom{n}{k} = \dfrac{n!}{k!\,(n-k)!}\)
Equivalent notation:
\(\binom{n}{k} = {}^nC_k = C(n,k)\)
Mean & Variance — §1.3
Mean (Expected Value)
\(E(X) = np\)
Average number of successes across all trials
Variance
\(\text{Var}(X) = np(1-p)\)
Also written as \(npq\), where \(q = 1-p\)
Standard Deviation
\(\sigma = \sqrt{np(1-p)}\)
Square root of the variance

§1.2 · Using Cumulative Probability Tables

What is a cumulative probability table?

The table gives you \(P(X \leq x)\) directly — the sum of all individual probabilities from 0 up to and including x. For \(X \sim B(n,p)\), tables are organised by n (rows) and p (columns). You can also use the Binomial CD function on your calculator.

01
Find your n. Locate the block of rows for your n value (e.g. n = 5).
02
Find your p. Move across to the column matching your probability of success.
03
Read the x-row. The value is P(X ≤ x). For non-direct forms, apply complement or subtraction.

Example 6 — Worked Solution

Using cumulative tables or Binomial CD on calculator

\(X \sim B(20,\, 0.4)\)
a \(P(X \leq 7) = ?\)
typeDirect lookup — already in ≤ form
tableFind n=20, p=0.40, row x=7
noteOr use Binomial CD on calculator with n=20, p=0.4, x=7
Answer\(P(X \leq 7) = 0.4159\)
b \(P(X < 6) = ?\)
key rule\(X\) takes only whole numbers, so \(X < 6\) means \(X \leq 5\)
rewrite\(P(X < 6)\) = \(P(X \leq 5)\)
tableFind n=20, p=0.40, row x=5
Answer\(P(X < 6) = P(X \leq 5) = 0.1256\)
c \(P(X \geq 15) = ?\)
rewriteTables only give \(P(X \leq x)\), so use the complement rule: \(P(X \geq 15) = \)\(1 - P(X \leq 14)\)
tableFind \(n=20,\, p=0.40\), row \(x=14\) → \(P(X \leq 14) = 0.9984\)
calc\(1 - 0.9984 = \)\(0.0016\)
Answer\(P(X \geq 15) = 1 - 0.9984 = 0.0016\)
