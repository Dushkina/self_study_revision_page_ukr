Формули інваріантності · Invariance Formulas — Пояснення та приклади

Як «переносити» табличні формули на складені функції

Завдяки теоремі про інваріантність формул інтегрування, кожну таблич­ну формулу можна застосувати не лише до аргументу \(x\), а й до будь-якої диференційовної функції \(u(x)\) — за умови, що поруч стоїть її похідна \(u'(x)\,dx = du\).

I
Логарифмічне правило · Logarithmic Rule
\(\displaystyle\int \frac{f'(x)}{f(x)}\,dx = \ln\left|f(x)\right| + C\)

Коли чисельник є похідною знаменника — результат одразу \(\ln|f(x)|\). Перевірте: \(\left(\ln|f|\right)' = \dfrac{f'}{f}\). Якщо чисельник відрізняється на константу — скоригуйте множник.

Приклад
\(\displaystyle\int\frac{2x}{x^2+3}\,dx\)
①\((x^2+3)' = 2x\) — чисельник є похідною знаменника ✓
✓\(= \ln(x^2+3) + C\)   (модуль не потрібен, бо \(x^2+3>0\))
Приклад з коригуванням
\(\displaystyle\int\frac{x}{x^2+1}\,dx\)
①\((x^2+1)' = 2x\), але в чисельнику лише \(x\) → домножаємо на \(\tfrac{1}{2}\)
②\(= \dfrac{1}{2}\displaystyle\int\frac{2x}{x^2+1}\,dx\)
✓\(= \dfrac{1}{2}\ln(x^2+1) + C\)
II
Корінь у знаменнику · Square-root Rule
\(\displaystyle\int \frac{f'(x)}{\sqrt{f(x)}}\,dx = 2\sqrt{f(x)} + C\)

Коли чисельник є похідною підкореневого виразу — результат \(2\sqrt{f(x)}\). Перевірка: \(\left(2\sqrt{f}\right)' = \dfrac{f'}{\sqrt{f}}\).

Приклад
\(\displaystyle\int\frac{\cos x}{\sqrt{\sin x}}\,dx\)
①\((\sin x)' = \cos x\) — чисельник є похідною підкореневого ✓
✓\(= 2\sqrt{\sin x} + C\)
Приклад з коригуванням
\(\displaystyle\int\frac{x}{\sqrt{x^2+5}}\,dx\)
①\((x^2+5)' = 2x\) → маємо \(x\), а потрібно \(2x\): домножаємо на \(\tfrac{1}{2}\)
✓\(= \dfrac{1}{2}\cdot 2\sqrt{x^2+5} + C = \sqrt{x^2+5}+C\)
III
Лінійна заміна · Linear Substitution
\(\displaystyle\int f(ax+b)\,dx = \frac{1}{a}\,F(ax+b) + C\)

Якщо аргумент функції — лінійний вираз \(ax+b\), просто застосовуємо таблич­ну формулу і ділимо результат на коефіцієнт \(a\) перед \(x\). Окремо: \(\int f(x+b)\,dx = F(x+b)+C\) (без ділення, бо \(a=1\)).

Приклад a
\(\displaystyle\int\sin(3x+1)\,dx\)
①\(\int\sin u\,du = -\cos u\), тут \(a=3\)
✓\(= -\dfrac{1}{3}\cos(3x+1) + C\)
Приклад b
\(\displaystyle\int e^{5x-2}\,dx\)
①\(\int e^u\,du = e^u\), тут \(a=5\)
✓\(= \dfrac{1}{5}e^{5x-2} + C\)
IV
Форма \(\frac{f'}{f}\) з константою · Rule 4
\(\displaystyle\int k\cdot\frac{f'(x)}{f(x)}\,dx\) — спробувати \(\ln|f(x)|\), потім перевірити та скоригувати

Якщо виразу є схожим на \(\tfrac{f'}{f}\), але з зайвим множником — спробувати \(\ln|f(x)|\), диференціювати та скоригувати константу.

Приклад
\(\displaystyle\int\frac{3x^2}{x^3-1}\,dx\)
①Спроба: \(\ln|x^3-1|\). Перевірка: \((\ln|x^3-1|)' = \dfrac{3x^2}{x^3-1}\) ✓
✓\(= \ln|x^3-1| + C\)
V
Форма \(f'\cdot[f]^n\) · Rule 5
\(\displaystyle\int k\,f'(x)\bigl[f(x)\bigr]^n\,dx\) — спробувати \(\bigl[f(x)\bigr]^{n+1}\), потім перевірити

Якщо підінтегральний вираз містить степінь функції, помножену на її похідну — спробувати \([f]^{n+1}\), диференціювати та скоригувати константу.

Приклад a
\(\displaystyle\int 2x(x^2+1)^4\,dx\)
①\((x^2+1)' = 2x\) — похідна стоїть поруч ✓
②Спроба: \((x^2+1)^5\). Перевірка: \(5(x^2+1)^4\cdot 2x\) — маємо зайвий множник 5
✓\(= \dfrac{1}{5}(x^2+1)^5 + C\)
Приклад b
\(\displaystyle\int\sin^3 x\cos x\,dx\)
①\((\sin x)' = \cos x\) — похідна стоїть поруч, \(n=3\)
②Спроба: \(\sin^4 x\). Перевірка: \(4\sin^3 x\cos x\) — скоригувати на \(\tfrac{1}{4}\)
✓\(= \dfrac{\sin^4 x}{4} + C\)
Основні властивості · Key Properties
1 · Лінійність · Linearity

Інтеграл від суми = сума інтегралів. Константу можна виносити.

\(\displaystyle\int\bigl[f(x)\pm g(x)\bigr]dx = \int f\,dx \pm \int g\,dx\)
\(\displaystyle\int k\cdot f(x)\,dx = k\int f(x)\,dx\)
2 · Перевірка · Verification

Будь-який невизначений інтеграл можна перевірити диференціюванням:

\(\displaystyle\left(\int f(x)\,dx\right)' = f(x)\)
\(\displaystyle d\!\left(\int f(x)\,dx\right) = f(x)\,dx\)
Приклади · Worked Examples
Приклад 1 · Формула 28
\(\displaystyle\int\frac{2x}{x^2+3}\,dx\)
①Чисельник є похідною знаменника: \((x^2+3)'=2x\) ✓формула 28: \(\int\frac{f'}{f}\,dx=\ln|f|\)
✓\(\displaystyle\int\frac{2x}{x^2+3}\,dx = \ln(x^2+3) + C\)модуль не потрібен, \(x^2+3>0\)
Приклад 2 · Формула 32 (інваріантність)
\(\displaystyle\int\sin(3x+1)\,dx\)
①Вигляд \(f(ax+b)\), де \(a=3,\;b=1\)формула 32
②\(\displaystyle\int\sin(3x+1)\,dx = \frac{1}{3}\cdot(-\cos(3x+1)) + C\)множимо на \(1/a=1/3\)
✓\(= -\dfrac{1}{3}\cos(3x+1) + C\)
Приклад 3 · Формула 29
\(\displaystyle\int\frac{\cos x}{\sqrt{\sin x}}\,dx\)
①Чисельник \(\cos x = (\sin x)'\) — похідна підкореневогоформула 29: \(\int\frac{f'}{\sqrt{f}}\,dx=2\sqrt{f}\)
✓\(\displaystyle\int\frac{\cos x}{\sqrt{\sin x}}\,dx = 2\sqrt{\sin x} + C\)
Приклад 4 · Формула 17
\(\displaystyle\int\frac{dx}{x^2+9}\)
①Форма \(\dfrac{1}{x^2+a^2}\) при \(a^2=9 \Rightarrow a=3\)формула 17
✓\(\displaystyle\int\frac{dx}{x^2+9} = \frac{1}{3}\arctan\frac{x}{3} + C\)
Приклад 5 · Інтегрування частинами
\(\displaystyle\int x\cos x\,dx\)
①Обрати: \(u=x\) (спрощується), \(dv=\cos x\,dx\)LIATE: алгебра перед trig
②\(du=dx,\quad v=\sin x\)
③\(\int u\,dv = uv - \int v\,du = x\sin x - \int\sin x\,dx\)формула по частинах
✓\(\displaystyle\int x\cos x\,dx = x\sin x + \cos x + C\)
Приклад 6 · Заміна змінної
\(\displaystyle\int e^{x^2}\cdot 2x\,dx\)
①Заміна: \(t = x^2 \Rightarrow dt = 2x\,dx\)підстановка
②\(\displaystyle\int e^t\,dt = e^t + C\)формула 6
✓\(\displaystyle\int e^{x^2}\cdot 2x\,dx = e^{x^2} + C\)повернутись до \(x\)
Приклад 7 · Тригонометрична тотожність
\(\displaystyle\int\sin^2 x\,dx\)
①Застосувати тотожність: \(\sin^2 x = \dfrac{1-\cos 2x}{2}\)властивість 5
②\(\displaystyle\int\frac{1-\cos 2x}{2}\,dx = \frac{1}{2}\int dx - \frac{1}{2}\int\cos 2x\,dx\)
③\(= \dfrac{x}{2} - \dfrac{1}{2}\cdot\dfrac{\sin 2x}{2}\)формула 9 + інваріантність
✓\(\displaystyle\int\sin^2 x\,dx = \frac{x}{2} - \frac{\sin 2x}{4} + C\)
