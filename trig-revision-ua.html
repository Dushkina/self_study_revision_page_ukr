Solving Basic Equations · Розв'язання базових рівнянь

sin x = a, cos x = a, tg x = a, ctg x = a

Загальні розв'язки (general solutions) для кожного випадку, k ∈ ℤ (k — ціле число).

sin x = a −1 ≤ a ≤ 1
\( x = (-1)^k \arcsin(a) + \pi k,\ k \in \mathbb{Z} \)
у градусах\( x = (-1)^k \arcsin(a) + 180^\circ k,\ k \in \mathbb{Z} \)
Особливі випадки · Special cases
sin x = 0
\(x = \pi k\)
\(x = 180^\circ k\)
sin x = 1
\(x = \tfrac{\pi}{2} + 2\pi k\)
\(x = 90^\circ + 360^\circ k\)
sin x = −1
\(x = -\tfrac{\pi}{2} + 2\pi k\)
\(x = -90^\circ + 360^\circ k\)
cos x = a −1 ≤ a ≤ 1
\( x = \pm \arccos(a) + 2\pi k,\ k \in \mathbb{Z} \)
у градусах\( x = \pm \arccos(a) + 360^\circ k,\ k \in \mathbb{Z} \)
Особливі випадки · Special cases
cos x = 0
\(x = \tfrac{\pi}{2} + \pi k\)
\(x = 90^\circ + 180^\circ k\)
cos x = 1
\(x = 2\pi k\)
\(x = 360^\circ k\)
cos x = −1
\(x = \pi + 2\pi k\)
\(x = 180^\circ + 360^\circ k\)
tg x = a  (або tan x) −∞ < a < +∞
\( x = \arctan(a) + \pi k,\ k \in \mathbb{Z} \)
у градусах\( x = \arctan(a) + 180^\circ k,\ k \in \mathbb{Z} \)
ctg x = a  (або cot x) −∞ < a < +∞
\( x = \text{arccot}(a) + \pi k,\ k \in \mathbb{Z} \)
у градусах\( x = \text{arccot}(a) + 180^\circ k,\ k \in \mathbb{Z} \)
Reciprocal Functions · Обернені функції

cosec x = a, sec x = a, ctg x = a

Ці рівняння зводяться (reduce) до базових через обернене (reciprocal) співвідношення: ставимо дріб \(\tfrac{1}{a}\) і розв'язуємо як звичайне рівняння.

cosec x = a a ∉ [−1, 1]

Крок 1 — переписати (rewrite) через sin:

\( \dfrac{1}{\sin x} = a \ \Rightarrow\ \sin x = \dfrac{1}{a} \)

Крок 2 — розв'язати як sin x = a:

\( x = (-1)^k \arcsin\!\left(\dfrac{1}{a}\right) + \pi k,\ k \in \mathbb{Z} \)
у градусах\( x = (-1)^k \arcsin\!\left(\dfrac{1}{a}\right) + 180^\circ k,\ k \in \mathbb{Z} \)
sec x = a a ∉ [−1, 1]

Крок 1 — переписати через cos:

\( \dfrac{1}{\cos x} = a \ \Rightarrow\ \cos x = \dfrac{1}{a} \)

Крок 2 — розв'язати як cos x = a:

\( x = \pm \arccos\!\left(\dfrac{1}{a}\right) + 2\pi k,\ k \in \mathbb{Z} \)
у градусах\( x = \pm \arccos\!\left(\dfrac{1}{a}\right) + 360^\circ k,\ k \in \mathbb{Z} \)
ctg x = a

Крок 1 — переписати через tan:

\( \dfrac{1}{\tan x} = a \ \Rightarrow\ \tan x = \dfrac{1}{a} \)

Крок 2 — розв'язати як tan x = a:

\( x = \arctan\!\left(\dfrac{1}{a}\right) + \pi k,\ k \in \mathbb{Z} \)
у градусах\( x = \arctan\!\left(\dfrac{1}{a}\right) + 180^\circ k,\ k \in \mathbb{Z} \)
Trigonometric Identities · Тригонометричні тотожності

Основні тотожності

Reciprocal Identities · Обернені співвідношення
reciprocal
\( \sec x = \dfrac{1}{\cos x} \)  (не визначено при cos x = 0)
\( \text{cosec } x = \dfrac{1}{\sin x} \)  (не визначено при sin x = 0)
\( \cot x = \dfrac{1}{\tan x} = \dfrac{\cos x}{\sin x} \)  (не визначено при tan x = 0)
Quotient Identities · Частинні співвідношення
quotient
\( \tan x = \dfrac{\sin x}{\cos x} \)
\( \cot x = \dfrac{\cos x}{\sin x} \)
\( \tan x \cdot \cot x = 1 \)
Pythagorean Identities · Тотожності Піфагора
pythagorean
\( \sin^2 x + \cos^2 x = 1 \)
\( 1 + \tan^2 x = \sec^2 x = \dfrac{1}{\cos^2 x} \)
\( 1 + \cot^2 x = \text{cosec}^2 x = \dfrac{1}{\sin^2 x} \)
Double Angle Identities · Формули подвійного кута
double angle
\( \sin(2x) = 2\sin x \cos x \)
\( \cos(2x) = \cos^2 x - \sin^2 x \)
\( \tan(2x) = \dfrac{2\tan x}{1 - \tan^2 x} \)
Power-Reduction Identities · Формули зниження степеня
half angle
\( \sin^2 x = \dfrac{1 - \cos 2x}{2} \)
\( \cos^2 x = \dfrac{1 + \cos 2x}{2} \)
Compound Angle Formulae · Формули додавання кутів
sum & difference
\( \sin(A+B) \equiv \sin A \cos B + \cos A \sin B \)
\( \sin(A-B) \equiv \sin A \cos B - \cos A \sin B \)
\( \cos(A+B) \equiv \cos A \cos B - \sin A \sin B \)
\( \cos(A-B) \equiv \cos A \cos B + \sin A \sin B \)
\( \tan(A+B) \equiv \dfrac{\tan A + \tan B}{1 - \tan A \tan B} \)
\( \tan(A-B) \equiv \dfrac{\tan A - \tan B}{1 + \tan A \tan B} \)
R-Form · Форма R sin(θ ± α)

Expressing a sin θ + b cos θ

Запис виразу a sin θ + b cos θ у вигляді R sin(θ ± α) або R cos(θ ± α) — зручно для розв'язання рівнянь і пошуку максимуму/мінімуму (max/min).

\( a\sin\theta \pm b\cos\theta = R\sin(\theta \pm \alpha) \)
\( a\cos\theta \pm b\sin\theta = R\cos(\theta \mp \alpha) \)
де \( R = \sqrt{a^2 + b^2} \), \(\quad \tan \alpha = \dfrac{b}{a} \)
Worked Example · Приклад розв'язання
Приклад · Розв'язання через обернену функцію
sec x = 2
① \(\sec x = 2\) вихідне рівняння
② \(\dfrac{1}{\cos x} = 2\) за визначенням sec x
③ \(\cos x = \dfrac{1}{2}\) обернена величина (reciprocal)
④ \(x = \pm \arccos\!\left(\dfrac{1}{2}\right) + 2\pi k\) формула розв'язання cos x = a
✓ \(x = \pm \dfrac{\pi}{3} + 2\pi k,\ k \in \mathbb{Z}\) arccos(1/2) = π/3
✓ \(x = \pm 60^\circ + 360^\circ k,\ k \in \mathbb{Z}\) те саме у градусах (degrees)
