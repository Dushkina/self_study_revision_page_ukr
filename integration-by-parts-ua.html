Інтегрування
частинами

Integration by Parts — потужний метод для інтегралів добутків (products) та трансцендентних функцій (transcendental functions).

Формула · Formula
\( \displaystyle \int u \, dv = u \cdot v - \int v \, du \)
Theory · Теорія

Коли і навіщо? · When and why?

Метод інтегрування частинами (integration by parts) застосовується, якщо підінтегральний вираз є добутком двох функцій, одна з яких спрощується при диференціюванні (differentiating), а інша — легко інтегрується (integrating).

Метод доцільно застосовувати для таких типів інтегралів:

Тип A — поліном × тригонометрія або поліном × експонента
u = поліном
\( \int P_n(x)\cos\alpha x\,dx, \quad \int P_n(x)\sin\alpha x\,dx, \quad \int P_n(x)e^{\alpha x}\,dx \)
Тип B — поліном × логарифм або зворотня тригонометрія
u = функція
\( \int P_n(x)\ln x\,dx, \quad \int P_n(x)\arcsin x\,dx, \quad \int P_n(x)\arctan x\,dx \)
Тип C — повторне інтегрування (cyclic)
повторне · cyclic
\( \int e^{\alpha x}\cos\beta x\,dx, \quad \int e^{\alpha x}\sin\beta x\,dx \)
Mind Map · Ментальна карта

Як вибрати u(x)? · How to choose u(x)?

вибір · choice of
\( u(x) = \, ? \)
Тип A · Type A
u(x) = поліном Pn(x)
(polynomial)
Коли? · When?
Підінтегральний вираз — добуток полінома та sin, cos або eαx
\(P_n(x) \cdot \sin\alpha x\)
Чому? · Why?
Поліном спрощується (simplifies) при диференціюванні: степінь зменшується на 1
\(P_n(x) \xrightarrow{d} P_{n-1}(x)\)
dv = ?
Тригонометрична або експонента частина → легко інтегрується
\(dv = \cos\alpha x\,dx\)
Приклади · Examples
\(\int x\sin x\,dx\) → \(u=x,\;dv=\sin x\,dx\)
\(\int(3x+4)e^{3x}dx\) → \(u=3x+4\)
Тип B · Type B
u(x) = ln x, arcsin x, arctan x
(transcendental function)
Коли? · When?
Підінтегральний вираз містить ln x, arcsin x або arctan x
\(P_n(x)\cdot\ln x\)
Чому? · Why?
Ці функції спрощуються при диференціюванні (похідна — раціональна)
\((\ln x)' = \tfrac{1}{x}\)
dv = ?
Поліном або dx — залишок виразу після u(x)
\(dv = x^2\,dx \Rightarrow v = \tfrac{x^3}{3}\)
Приклади · Examples
\(\int\ln x\,dx\) → \(u=\ln x,\;dv=dx\)
\(\int x^2\arctan x\,dx\) → \(u=\arctan x\)
Тип C · Cyclic
\(\int e^{\alpha x}\sin\beta x\,dx\) та \(\int e^{\alpha x}\cos\beta x\,dx\)

Можна взяти будь-яку з двох функцій як u(x). Головне — не змінювати вибір при повторному інтегруванні! Після двох ітерацій отримаємо рівняння відносно I (cyclic equation) → розв'язуємо алгебраїчно.

Algorithm · Алгоритм

Покроковий метод · Step-by-Step

01

Визначити тип інтеграла · Identify the type

Подивитися на структуру підінтегрального виразу: це тип A, B чи C? (polynomial × trig/exp, або ln/arcsin/arctan, або cyclic)

02

Вибрати u(x) та dv · Choose u(x) and dv

Відповідно до типу: A → u = поліном; B → u = ln/arcsin/arctan; C → u = будь-яка (і не змінювати).

Правило LIATE: Log → Inverse trig → Algebraic → Trig → Exp
03

Знайти du та v · Find du and v

Диференціювати u → отримати du. Інтегрувати dv → отримати v (константу не писати).

\(du = u'(x)\,dx, \quad v = \int dv\)
04

Підставити у формулу · Apply the formula

Записати \(\int u\,dv = u\cdot v - \int v\,du\) та спростити новий інтеграл.

05

Обчислити залишковий інтеграл · Evaluate remaining integral

Якщо потрібно — повторити метод (тип A з поліномом вищого ступеня або тип C).

Для типу C: перенести I в одну частину та розділити!
06

Додати константу C · Add constant of integration

Не забути +C наприкінці для невизначеного інтеграла (indefinite integral).

Worked Examples · Розібрані приклади
Приклад 1 · Тип A — поліном × синус
\(\int x \sin x \, dx\)
① Тип A: \(u = x,\quad dv = \sin x\,dx\) поліном × тригонометрія
② \(du = dx;\quad v = \int\sin x\,dx = -\cos x\) диференціюємо u, інтегруємо dv
③ \(\int x\sin x\,dx = x\cdot(-\cos x) - \int(-\cos x)\,dx\) підставляємо у формулу
④ \(= -x\cos x + \int\cos x\,dx = -x\cos x + \sin x\) простий залишковий інтеграл
✓ \(\int x\sin x\,dx = -x\cos x + \sin x + C\) додаємо константу
Приклад 2 · Тип B — логарифм
\(\int \ln x \, dx\)
① Тип B: \(u = \ln x,\quad dv = dx\) логарифм спрощується при диференціюванні
② \(du = \dfrac{dx}{x};\quad v = \int dx = x\) find du and v
③ \(\int\ln x\,dx = x\ln x - \int x\cdot\dfrac{dx}{x} = x\ln x - \int dx\) apply the formula
✓ \(\int\ln x\,dx = x\ln x - x + C = x(\ln x - 1) + C\) спрощуємо
Приклад 3 · Тип C — повторне інтегрування (cyclic)
\(\int e^x \sin 2x \, dx\)
① Тип C: \(u = e^x,\quad dv = \sin 2x\,dx\) вибираємо і не змінюємо!
② \(du = e^x dx;\quad v = -\dfrac{\cos 2x}{2}\) перше інтегрування
③ \(I = -\dfrac{e^x\cos 2x}{2} + \dfrac{1}{2}\int e^x\cos 2x\,dx\) застосовуємо формулу
④ Знову: \(u=e^x,\;dv=\cos 2x\,dx \Rightarrow v=\dfrac{\sin 2x}{2}\) повторне інтегрування частинами!
⑤ \(I = -\dfrac{e^x\cos 2x}{2} + \dfrac{1}{2}\!\left(\dfrac{e^x\sin 2x}{2} - \dfrac{1}{2}I\right)\) I знову з'являється справа!
⑥ \(I = -\dfrac{e^x\cos 2x}{2} + \dfrac{e^x\sin 2x}{4} - \dfrac{1}{4}I\) розкриваємо дужки
⑦ \(\dfrac{5}{4}I = -\dfrac{e^x\cos 2x}{2} + \dfrac{e^x\sin 2x}{4}\) переносимо \(-\tfrac{1}{4}I\) вліво
✓ \(I = \dfrac{e^x}{5}(\sin 2x - 2\cos 2x) + C\) ділимо обидві частини на \(\tfrac{5}{4}\)
Приклад 4 · Тип C — класичний цикл
\(\int e^x \cos x \, dx\)
① \(u = e^x,\quad dv = \cos x\,dx \Rightarrow du = e^x dx,\;v = \sin x\) перший крок
② \(I = e^x\sin x - \int e^x\sin x\,dx\) після першого застосування формули
③ Для \(\int e^x\sin x\,dx\): \(u=e^x,\;dv=\sin x\,dx \Rightarrow v=-\cos x\) повторно — той самий вибір u!
④ \(I = e^x\sin x - \bigl(-e^x\cos x + \int e^x\cos x\,dx\bigr)\) підставляємо
⑤ \(I = e^x\sin x + e^x\cos x - I \;\Rightarrow\; 2I = e^x(\sin x + \cos x)\) cyclic equation! Збираємо I
✓ \(\int e^x\cos x\,dx = \dfrac{e^x(\sin x + \cos x)}{2} + C\) ділимо на 2
